# Ulko — Know before you go.

> How long can you safely stay outside — in what you're wearing right now?

A hybrid AI system that translates Arctic weather into personalized safety metrics.
Built for Arktinen Hehku 2026, Salla Wilderness Park, Lapland.

## Features

- **AI Gear Coach** — type your outfit, get a thermal insulation score, gear summary, and gap warnings
- **Physics Rule Engine** — Wind Chill Index + humidity + activity → safe exposure time
- **Ticking countdown timer** — pulses when you're in the danger zone
- **SHAP Explainability** — see exactly why your risk level is what it is
- **24-Hour Window of Safety** — color-coded forecast so you know when to go out

## Setup

1. Clone this repo
2. Open `index.html` in your browser — done

## Deploy to GitHub Pages

1. Push to GitHub
2. Settings → Pages → Source: main branch / root
3. Live at `https://... .github.io/ulko`

## Stack

Pure HTML / CSS / JavaScript — no framework, no build step, no dependencies.
Anthropic Claude API · OpenWeatherMap API

## Disclaimer

Ulko is a decision support tool only. It is not a medical device.
Always exercise personal judgment and err on the side of caution in cold environments.
